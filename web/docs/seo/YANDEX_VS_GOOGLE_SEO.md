1. Go to [Yandex Webmaster](https://webmaster.yandex.ru/)

## üîç Key Differences

### Google Search Console

- **Interface**: English (mostly)
- **Focus**: Global SEO
- **Features**: Core Web Vitals, Mobile Usability, Rich Results
- **Indexing**: Usually faster (hours to days)

### Yandex Webmaster

- **Interface**: Russian (can switch to English)
- **Focus**: Russian market SEO
- **Features**: Region targeting, Yandex-specific structured data
- **Indexing**: Can be slower (days to weeks)
- **Special**: Yandex.Metrica integration

## üö® Critical for URL Preservation

**Both tools need to see the same URLs:**

When you switch servers:

1. **Google Search Console**
   - Use "Change of Address" tool (if migrating domains)
   - Or just add new property and submit sitemap
   - Monitor for 404 errors

2. **Yandex Webmaster**
   - Use "Site Moving" tool (if migrating domains)
   - Or just add new site and submit sitemap
   - Monitor for indexing errors

**Both will check if your indexed URLs still work!**
